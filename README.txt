Всем привет!

Меня зовут Аркадий, и это программа на Python, написанная мной в рамках ВКР по теме "Изоморфизм эллиптических кривых над конечным полем в криптографии".

Программа умеет:

1. Находить все неизоморфные эллиптические кривые над выбранным полем Fp (конечное простое поле).
2. Вычислять структуру группы точек над Fp.
3. Находить все изоморфные кривые для выбранной кривой.

Для работы программы необходимо скачать репозиторий и запустить файл "Isomorphism-of-elliptic-curves-master\dist\main.exe". Исходный код находится в "Isomorphism-of-elliptic-curves-master\main.py".
