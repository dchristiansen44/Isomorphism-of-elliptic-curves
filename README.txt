Всем привет!

Меня зовут Аркадий, и это программа на Python, написанная мною в рамках ВКР по теме "Изоморфизм эллиптических кривых над конечным полем в криптографии".

Программа умеет:

1. Находить все неизоморфные эллиптические кривые над выбранным полем Fp (конечное простое поле).
2. Вычислять структуру группы точек над Fp.
3. Находить все изоморфные кривые для выбранной кривой.

Для работы программы необходимо скачать все файлы репозитория и запустить файл "main.py". На устройстве должен быть предустановлен Python 3, а также PyQt5 — набор расширений графического фреймворка Qt для языка программирования Python.
